Turn your Epic order history into an ITAD collection backup

Resolves each game to its real ITAD id (by Epic offer id, falling back to title) and writes a collection.backup.json you import yourself. No login, no API key needed, no server — lookups go straight to ITAD's public endpoints from your browser.

  2. Open DevTools (F12) → Console tab.
  3. Paste the script below, press Enter, let it finish.
  4. It copies the JSON to your clipboard and downloads epic-orders.json.
