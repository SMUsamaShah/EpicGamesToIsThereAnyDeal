Epic only shows ten orders per page. This script walks every page for you and saves one combined file — no clicking Next and copying from the Network tab.

  1. Sign in to Epic and open epicgames.com/account/transactions.
  2. Open DevTools (F12) and go to the Console tab.
  3. Paste the script below, press Enter, and let it finish.
  4. It copies the JSON to your clipboard and downloads epic-orders.json. Use either below.
