# EpicGamesToIsThereAnyDeal
Import list of games on Epic store on ITAD (isthereanydeal.com)

## Step 1

- Go to https://accounts.epicgames.com/account/transactions/purchases?lang=en-US&productName=egs
- Open browser dev console
- Copy the contents of script epic-orders-export.js into dev console and run. It will go over all the pages and download a epic-export.json
